# 小程序开发 Let's do

2020.2.28建

## 功能介绍

- 可设置备忘录，上课前30分钟提醒上课。

- 提供了天院特色地图。

- 为用户提供部分常见的可选择的食物，可手动输入并保存备选物品。

- 在用户选择出的若干个备选物品里随机选择一个。
